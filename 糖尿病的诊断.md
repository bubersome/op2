## 怎样诊断糖尿病?

诊断糖尿病的常用检验方法有四种:

- 随机（或偶然）血浆葡萄糖检测
- 空腹血浆葡萄糖检测 
- 口服葡萄糖耐量试验
- A1c (糖化血红蛋白)

## 随机（或偶然）血浆葡萄糖

血浆葡萄糖检测，是用来检查您的血液循环里面有多少糖份/葡萄糖。“随机”和“偶然”的意思，只不过是说您可以在任何时间到化验室取血。不管您吃没吃东西，都不影响化验结果。如果有一次血浆葡萄糖检测的结果高于或等于200 毫克每分升(mg/dl)，那就提示您可能患有糖尿病。为了确诊，您必须另找一天再做一次化验。这次化验可以是空腹取血，也可以是另一次随机血糖。

## 空腹血浆葡萄糖检测

这项检测简便易行，在您禁用食物和饮料（除了水之外）至少8小时后进行。禁食后血浆葡萄糖水平的正常值是在60 和 99 mg/dl之间。只有在两次不同时间的空腹血浆葡萄糖检测都高于或等于126时，才能确诊糖尿病。

## 口服葡萄糖耐量试验

有一种可能性是，糖尿病患者—即使是有症状的患者—也可能空腹血浆葡萄糖检测正常。如果您碰到这种情况，医生就会请您再次禁用食物和饮料（除了清水之外）至少8小时，然后喝上一杯葡萄糖溶液（一般是75克）。在喝下葡萄糖溶液之前和之后两小时后抽血。这项检测称为口服葡萄糖耐量试验。

空腹血浆葡萄糖水平通常小于100 mg/dl。如果在100 – 126 mg/dl 之间，表明你有糖尿病前期。如果大于126 mg/dl，则可以诊断糖尿病。

喝下葡萄糖溶液两小时后的正常血糖应该在140 mg/dl之下。如果在140 – 199 mg/dl 之间，表明你有糖尿病前期。如果高于或等于200 mg/dl，就表明您患有糖尿病。

## A1c （糖化血红蛋白）检测

[A1c （糖化血红蛋白）检测 ](https://dtc.ucsf.edu/zh-hans/糖尿病分型/2型糖尿病/糖尿病的治疗/糖尿病治疗目标/了解平均血糖值/)表明您最近三个月的平均血糖水平。美国糖尿病学会推荐用[A1c （糖化血红蛋白）检测 ](https://dtc.ucsf.edu/zh-hans/糖尿病分型/2型糖尿病/糖尿病的治疗/糖尿病治疗目标/了解平均血糖值/)来诊断糖尿病和糖尿病前期。按照美国糖尿病学会的标准，糖化血红蛋白大于6.5%，可以考虑诊断为糖尿病；糖化血红蛋白在5.7% – 6.4% 之间，可诊断糖尿病前期，正常值小于5.6%。

**一点注意：**

1. ***TA1C检测是间接测量血糖的平均水平，有可能并不准确。例如，某人有某种类型的血红蛋白突变（血红蛋白结构的变化）、严重贫血（红细胞计数低）、正在接受输血、或正在使用增加血红细胞生成的药物，A1C检测都可能不准确。***
2. ***使用A1C来诊断糖尿病，目前是有争议的，只有美国糖尿病协会认可使用A1C来诊断糖尿病。***

### 此表列出了美国糖尿病协会的糖尿病诊断标准：

|                                            | 正常            | 糖尿病前期        | 糖尿病              |
| ------------------------------------------ | --------------- | ----------------- | ------------------- |
| 空腹血浆葡萄糖                             | <100毫克/分升   | 100-125 毫克/分升 | 126 毫克/分升或更高 |
| 口服葡萄糖耐量试验, 饮用含高糖饮料后2 小时 | < 140 毫克/分升 | 140-199毫克/分升  | 200毫克/分升或更高  |
| 随机或即刻血糖和症状                       |                 |                   | 200毫克/分升或更高  |
| 糖化血红蛋白                               | <5.7%           | 5.7-6.4%          | 6.5%或更高          |
